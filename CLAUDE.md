# Apeiron Product Brain

## Auto-commit rule

Whenever changes are made to `index.html`, automatically:

1. `git add index.html`
2. `git commit -m "<descriptive message summarizing the changes>"` — the commit message should clearly describe what was changed (e.g., "Update product card layout", "Add carrier logo section").
3. `git push` to GitHub.

Do this immediately after every edit to `index.html` without waiting to be asked.
